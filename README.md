# Patrimônio OS

Dashboard financeiro pessoal em um único arquivo HTML — sem dependências, estilo iOS.

**Acesse:** https://helionsc.github.io/patrimonio-os/

## Recursos
- Patrimônio total com anel de progresso para a próxima meta
- Carteira de investimentos (ativos, análise com gráficos, projeções)
- Cartões de crédito com divisão de fatura entre titular e adicional
- Metas, milhas e cálculo de salário líquido (INSS/IRRF)
- Insights automáticos, score financeiro e modo privacidade
- Sincronização em nuvem (Supabase) + export/import JSON
- Funciona offline; "Adicionar à Tela de Início" instala como app (PWA)

## Uso
Abra a URL no Safari (iPhone) ou em qualquer navegador. Os dados ficam salvos localmente no dispositivo e podem ser sincronizados pela aba **Mais**.
